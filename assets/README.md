# Assets Directory

This directory contains all static assets for the Spotnere Vendor app.

## Fonts

Place the following Parkinsans font files in `assets/fonts/`:
- Parkinsans-Light.ttf
- Parkinsans-Regular.ttf
- Parkinsans-Medium.ttf
- Parkinsans-SemiBold.ttf
- Parkinsans-Bold.ttf
- Parkinsans-ExtraBold.ttf

## Icons

Place the following icon files in `assets/icons/`:
- adaptive-icon.png (Android adaptive icon foreground)
- ios-dark.png (iOS dark mode icon)
- ios-light.png (iOS light mode icon)
- ios-tinted.png (iOS tinted icon)
- splash-icon.png (Splash screen icon)

## Root Assets

Place the following files in `assets/`:
- icon.png (Main app icon, 1024x1024px)
- favicon.png (Web favicon, optional)

## Note

These assets are required for the app to build and run properly. Make sure all files are present before building the app.
